Work

Flashback
Flashback

My team was able to accomplish the creation of a physical prototype, and app concept for the flashback project. The design goal of this project was to create a device with an accompanying phone app that users could use for education purposes, and strengthen the mind through memorization games. Some games found in our prototype range from matching light patterns or matching directional audio cues, and as the games go on, the patterns become longer, and challenge user to get to that next level. Another note is that flashback features haptic vibration to indicate to users if they were successful or not.

Proximate
Proximate

My team was able to accomplish the creation of a physical prototype, and app concept for the proximate project. The design goal of this project was to create a device that could be very flexible in how it was used, to appeal to a wide audience, and support people with disabilities and impairments. Along side the physical prototype, we designed an app that would allow users to customize how the device worked. For example, if a user with visual impairments was crossing the road, and a car is coming, the device would detect danger, and depending how the users chosen settings from the app, the device would vibrate, play a noise, or both, to indicate to the user to stop and wait.
